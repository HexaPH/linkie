## Issue / Bug Reporting
If you find an issue, please report it to me in the Issues tab on GitHub. Or, you can report it on our [Discord server](https://discord.gg/FBfWyvajcK).
> **Note:** I do prefer GitHub.

## Think Linkie's missing something?
